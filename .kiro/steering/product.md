# Product Overview

PayMe Protocol is a React Native mobile application demonstrating pixel-perfect iOS-native design patterns. This is a demo app showcasing Apple Human Interface Guidelines implementation with no backend dependencies.

## Core Purpose

Educational reference implementation for iOS-native design in React Native, featuring:
- iOS-native UI components (San Francisco typography, system colors, native navigation)
- Complete design system with reusable components
- Biometric authentication and haptic feedback integration
- Full accessibility support (VoiceOver, Dynamic Type)

## Key Features

- Onboarding flow with centered icon and CTA
- Dashboard with balance card and transaction list
- Transaction detail view with action buttons
- Settings screen with iOS-style toggles
- Mock data layer (no real backend required)

## Design Philosophy

This app prioritizes iOS design fidelity over platform conventions. iOS design patterns are intentionally applied on both iOS and Android platforms (no Material Design components). The app follows Apple's Human Interface Guidelines strictly, including typography, colors, spacing, navigation patterns, and animations.

## Target Platforms

- Primary: iOS (iPhone and iPad)
- Secondary: Android (using iOS design language)
- Web: Basic support via Expo web
